# P10 Pick

Mobile-first fantasy Formula 1 web app where players predict:

- 10th-place finisher
- First retirement
- Fastest lap

Features include automated race scoring via official results, custom points tables, leagues, leaderboards, and small bonus points (e.g., fastest pit stop, most overtakes).

Repo layout:

- /client – React PWA (Vite for local dev)
- /api – Express API (Node for local dev)
- /supabase – SQL migrations and seeds
- /docs – PRD, site map, OpenAPI, deployment notes
